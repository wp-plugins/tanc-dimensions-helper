Adds an updating viewport width to the WordPress admin bar.

== Description ==

This is a helper plugin that adds the width of the browser viewport to the WordPress admin bar. It also auto-updates as you resize the browser width.

Uses:

*   Checking media query breakpoints behave properly
*   Checking to see how your site looks at specific widths

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the whole folder `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy!
